500 Main Page for Spring 2025

Published

2024-12-24

Welcome! PQHS 500 (also called CRSP 500) is taught by Professor Thomas Love in the Department of Population and Quantitative Health Sciences at Case Western Reserve University. Everything that Professor Love will provide to help you with the course will appear on this website by the start of the semester.

This year’s class will be unusual.

  • Thanks to Dr. Love’s recovery from a broken leg and ankle surgery, this year’s 500 class will be held remotely before Spring Break, and then in person starting March 20. See the Course Calendar for more details.

Everything You Need Is Linked Here

The menu bar includes links to just about everything you’ll need this semester, including

  • the course syllabus
  • the course calendar, which provides the final word on all class sessions and deadlines, and also provides links to the materials used in each class session.
  • course assignment information, including labs, the Observational Studies in Action (OSIA) presentations of studies from the literature, and a semester-long project.
  • information on software, including installation of R and RStudio, key R packages, downloading data (and code) you’ll need for the class.
  • links to additional Sources (articles, etc.) that we’ll be reading this semester, or that I think might be helpful to you.
  • a link to the Canvas system (CWRU log-in required) we use for turning in some assignments and posting links to recordings,
  • information about where you can get help with the course and contact Dr. Love.

What Should I Do Before Class Begins?

In addition to exploring what’s available on this website, you should …

  1. Buy the book. During the course, we will read Paul Rosenbaum’s book Causal Inference, which is available as an e-book or in paperback for under $15. Please buy the book and get started reading it as soon as you can.

  2. Once you have registered for the course, please visit our Welcome to 500 survey. You’ll need to log into Google via CWRU to access the survey, which should take about 10 minutes to complete. Please complete the survey as soon as possible.

  3. Download the software. Install R and RStudio and some necessary R packages on a computer you can control throughout the semester. Details are available here.
